One Command Combiner

As of now, the command combiner is very fragile. This is because I am too stupid to make it better. Nevertheless, it is still pretty decent. See below on how to use this monstrosity.






A new line = a new command





The above will put down a repeat command block facing up with the command 'say a'.
A chain will be placed on top of it with the command 'say b'.
On top of that will be a chain conditional block that has 'say c'.
On top of that will be a chain command block with 'say d' (A line without a type prefix will default to C:).
++z++ - A Z offset will be applied. Meaning the commands will go to a new row. (~ ~ ~ goes to ~ ~ ~+1)!
On top of that will be a repeating conditional block with 'say ree'.
On top of that will be a regular impulse block with 'say test'.
On top of that will be an impulse conditional with say 'smee'.
Before all of this, a command 'scoreboard objectives add die dummy' will be done once (N: signifies an initial command).




As this is fragile, please be conscious of your syntax. Common mistakes include:

I know, it looks just fine. But the R: say a must be R:say a. (That 1 space makes a whole difference)
